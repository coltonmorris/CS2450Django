The database is sqlite3, you must have that downloaded.

In Linux: sudo apt-get install sqlite3
In windows: https://www.sqlite.org/download.html

Reference this webiste for information on syntax and commands, it is very straight forward, and has examples for every command you would need to use.
	http://www.tutorialspoint.com/sqlite/index.htm

We can use sqlite3 very easily with python, reference this site for more information. Simply import sqlite3, setup a connection, and execute commands.
	http://docs.python.org/2/library/sqlite3.html
